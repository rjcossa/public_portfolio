Telecommunications Background

Why Telecommunications?

I chose to work on the telecommunications sector because it is one of the fastest growing sectors in Mozambique and also because working in telecommunications is one of the biggest challenges as it changes every day! When i first started working in telecommunications i remember that there was no documentation as it is a very closed sector so it really took hands on experience to learn everything that i have learnt to date.

Biggest Challenges?

As i started working on telecommunications at Vodacom Mozambique, i was given many high performance systems to manage of which one stands out because of it's mission critical status and system complexity: The HLR! In a communications network the HLR (Home Location Register) is the central repository for the mobile network's subscriber and routing information! If an HLR dies the whole network goes down and no calls, sms' or USSD messages can be completed.
